# Student Success Predictor

This project predicts student success based on provided data using machine learning techniques.

## Project Structure
- **main.py**: Main script for running the prediction model.
- **requirements.txt**: List of required Python packages.
- **student_data.csv**: Dataset containing students' past informations.
- **understanding_data.txt**: Notes and explanations about the dataset.

## Getting Started
1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/Student_Success_Predictor.git
   cd Student_Success_Predictor
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```
   
3. **Run the main script**
   ```bash
   python main.py
   ```

## Motivation
Student success prediction helps educators and institutions identify at-risk students early and provide targeted interventions to improve outcomes.

## Features
- Data preprocessing and cleaning
- Exploratory data analysis
- Machine learning model for prediction
- Easy-to-modify code for experimentation

## Data
- The dataset (`student_data.csv`) contains features relevant to student performance.
- See `understanding_data.txt` for more details about the data fields.

## Requirements

- See `requirements.txt` for all dependencies.

## Usage
Modify `main.py` as needed to experiment with different models or features.
